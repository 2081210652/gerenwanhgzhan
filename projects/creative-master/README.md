// 伪代码
card.onMouseEnter = () => {

## 📝 八、开发注意事项

### 8.1 代码规范

- 所有 CSS 类名使用 BEM 命名规范
- JavaScript 使用 ES6+ 语法
- 所有交互都要有加载状态和错误处理
- 注释清晰，说明每个函数的用途

### 8.2 性能优化

- 图片使用懒加载
- 视频使用预加载 (preload="metadata")
- 使用 CSS 动画代替 JavaScript 动画
- 合理使用防抖和节流

### 8.3 兼容性

- 支持 Chrome、Firefox、Safari、Edge 最新版本
- 移动端适配 (响应式设计)
- 降级处理 (视频不支持时显示静态图)
